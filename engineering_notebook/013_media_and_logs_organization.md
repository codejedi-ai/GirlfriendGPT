# Media and Logs Organization

**Date:** 2026-03-12  
**Status:** ✅ Complete  

## Overview

All generated media and logs are now organized in dedicated folders under `~/.gfgpt/` for easy access and management.

---

## Directory Structure

```
~/.gfgpt/
├── config.json                 # Active configuration
├── state.json                  # Runtime state (PID, etc)
├── logs/                       # Log files
│   └── gateway.log            # Gateway server logs
├── media/                      # Generated media
│   ├── img/                   # Generated images
│   ├── video/                 # Generated videos
│   └── audio/                 # Generated audio
└── templates/                  # Templates
    ├── config.json            # Config template
    ├── tools.md               # System prompt template
    └── personalities/         # Personality templates
```

## Logs

### Location
`~/.gfgpt/logs/gateway.log`

### What's Logged
- Gateway startup/shutdown
- Client connections/disconnections
- Agent reloads (hot-reload)
- Errors and warnings
- Message processing

### View Logs

**Tail (live):**
```bash
tail -f ~/.gfgpt/logs/gateway.log
```

**Last 100 lines:**
```bash
tail -n 100 ~/.gfgpt/logs/gateway.log
```

**Search:**
```bash
grep "ERROR" ~/.gfgpt/logs/gateway.log
grep "reload" ~/.gfgpt/logs/gateway.log
```

### Log Format
```
2026-03-12 23:22:15,123 - gateway - INFO - Starting AI Influencer Gateway
2026-03-12 23:22:15,456 - gateway - INFO - Agent: Sandra
2026-03-12 23:22:15,789 - gateway - INFO - Config watcher started
2026-03-12 23:22:20,123 - gateway - INFO - Client connected: abc123
```

## Access Media Programmatically

### Python API

```python
from src.config import ConfigManager

# Get media directories
img_dir = ConfigManager.get_media_dir("img")
video_dir = ConfigManager.get_media_dir("video")
audio_dir = ConfigManager.get_media_dir("audio")

# List generated images
for img_file in img_dir.glob("*.png"):
    print(f"Image: {img_file.name}")

# Get log file
log_file = ConfigManager.get_log_file("gateway")
print(f"Logs: {log_file}")
```

### Command Line

```bash
# List all media
find ~/.gfgpt/media -type f

# List images only
ls -la ~/.gfgpt/media/img/

# List videos
ls -lh ~/.gfgpt/media/video/

# Check log size
du -sh ~/.gfgpt/logs/
```

## Cleanup

### Rotate Logs

```bash
# Compress old logs
gzip ~/.gfgpt/logs/gateway.log

# Clear logs (be careful!)
> ~/.gfgpt/logs/gateway.log

# Delete logs older than 7 days
find ~/.gfgpt/logs -name "*.log" -mtime +7 -delete
```

### Clean Media

```bash
# Delete images older than 30 days
find ~/.gfgpt/media/img -type f -mtime +30 -delete

# Delete all media (careful!)
rm -rf ~/.gfgpt/media/*

# Check media size
du -sh ~/.gfgpt/media/
```

## Backup

### Backup Everything

```bash
# Create backup
tar -czf gfgpt-backup-$(date +%Y%m%d).tar.gz ~/.gfgpt/

# Restore backup
tar -xzf gfgpt-backup-20260312.tar.gz -C ~/
```

### Backup Config Only

```bash
cp ~/.gfgpt/config.json ~/.gfgpt/config.backup.json
```

### Backup Media Only

```bash
tar -czf gfgpt-media-backup.tar.gz ~/.gfgpt/media/
```

## Configuration

### Custom Media Location

Edit `~/.gfgpt/config.json`:

```json
{
  "media_dir": "/path/to/custom/media",
  "logs_dir": "/path/to/custom/logs"
}
```

### Auto-Cleanup Settings

Future enhancement: Add to config:

```json
{
  "auto_cleanup": {
    "logs_max_age_days": 7,
    "media_max_age_days": 30,
    "max_storage_gb": 10
  }
}
```

## Troubleshooting

### Logs Not Being Created

**Check:**
```bash
# Verify logs folder exists
ls -la ~/.gfgpt/logs/

# Check permissions
chmod 755 ~/.gfgpt/logs/

# Check gateway output
gfgpt gateway start
```

### Media Not Saved

**Check:**
```bash
# Verify media folders exist
ls -la ~/.gfgpt/media/

# Check disk space
df -h ~/.gfgpt/

# Check tool configuration
cat ~/.gfgpt/config.json
```

### Disk Space Full

**Clean up:**
```bash
# Find large files
find ~/.gfgpt -type f -size +100M

# Delete old media
find ~/.gfgpt/media -type f -mtime +30 -delete

# Compress logs
gzip ~/.gfgpt/logs/*.log
```
